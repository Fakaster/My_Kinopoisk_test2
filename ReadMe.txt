В качестве курсовой работы был выбран сервис кинопоиск
Данная БД предназначена для хранения медиафайлов, информации об актерах и пользователях ресурса Кинопоиск
Для удобной работы с БД прописаны основные Select запросы которые могут понадобится для формирования итоговых данных ддя пользователей
Так же созданы представления total_films и total_user хранящие в себе всю информацию которая используется при просмотре профиля пользователя, или просмотре профиля фильма
